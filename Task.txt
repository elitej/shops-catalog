Разработать на Java с использованием 3-х звенной архитектуры приложение для
отображения каталога магазинов с имеющимся у них ассортиментом товаров.

структура данных в БД:

 Каждый магазин в каталоге описывается:
    - ID -- уникальный идентификатор магазина.
    - Name -- уникальное название магазина.
    - Address -- адрес магазина.

 Каждый товар  описывается:
    - ID -- уникальный идентификатор товара.
    - Name -- уникальное название товара.
    - Producer -- производитель товара.


Приложение предоставляет следующие функции:
    - Изображение каталога магазинов с соответствующим ассортиментом товаров.


Используемые технологии:
Тонкий клиент(современный браузер), Tomcat, java(servlets, jsp), БД(любая)

JDK и библиотеки.
-----------------

    Приложение должно компилироваться и запускаться с использованием SUN
    JDK 1.6 и выше.


Поддерживаемость кода.
----------------------

    При разработке приложения, необходимо руководствоваться теми же
    соображениями, что и при разработке коммерческого кода, который будет
    долгое время развиваться, масштабироваться и поддерживаться (в том числе, другими
    разработчиками).

    В частности:

        - Должна обеспечиваться хорошая читаемость и поддерживаемость кода.

        - Код должен соответствовать хорошо зарекомендовавшим себя
          практикам, которые вы обычно применяете.
